# Changelog

## 0.1.0 (2026-06-11)

- Initial release.
- Syntax highlighting for all `.jrg` file constructs:
  - Metadata `~~~~~` separator and key:value pairs.
  - Headings (`#` through `######`).
  - Target links (`=> ...`).
  - Action buttons (`! ...`), input fields (`? ...`), image tags (`@ ...`).
  - List items (`- ...`), quotes (`> ...`).
  - Horizontal rules (`---`).
  - Table rows (`| ... |`).
  - Fenced code blocks with optional language tag.
  - Line comments (`//`), block comments (`/* */`).
  - Quoted strings (double, single, backtick) and numeric literals.
- Language configuration with line comments and auto-closing bracket pairs.
